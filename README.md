# IDL-Definitions
IDL Definitions for your editor, currently a collection of git submodules.

## Webstorm
1. Clone to local machine
1. Preferences (`⌘,`) → Languages & Frameworks → JavaScript → Libraries → Add
1. Click the `+` button (`⌘N`)
1. Select this directory
1. Enjoy
